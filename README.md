Expression Calculation
======================

Ruby Practice - Simple Calculation

Run `ruby runner.rb` in console

You can input some expressions like below:

- ( 2 + ( ( 4 + 6 ) * (9 * 2) - ( 5 - 1)))
- ( 2 + ( ( 4 + 6 ) * sqrt(5)) / 2 )

Run `bundle install` then `cd spec`

Run `rspec expression_spec.rb` for testing
